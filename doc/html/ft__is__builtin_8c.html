void clean_exit ( char ** cmd,
char *** envp )

Gère la sortie propre du shell avec sauvegarde de l'environnement.

Paramètres
cmdTableau des arguments de la commande exit
envpPointeur vers le tableau des variables d'environnement

Cette fonction gère plusieurs cas de la commande exit :

  • Sans argument : sortie avec code 0
  • Un argument numérique : sortie avec ce code (modulo 256)
  • Un argument +0 ou -0 : sortie avec code 0
  • Argument invalide : sortie avec code 2
  • Plusieurs arguments : sortie avec code 1 (sauf si le premier est alphabétique -> code 2)

Sauvegarde toujours l'environnement avant la sortie via save_env().

int execute_builtin ( char ** cmd,
char *** envp )

Exécute la commande builtin appropriée.

Paramètres
cmdTableau des arguments de la commande
envpPointeur vers le tableau des variables d'environnement
Renvoie
int Code de retour de la commande exécutée

Cette fonction dispatche l'exécution vers la fonction builtin appropriée en fonction du premier argument (cmd[0]). Gère :

  • export : création/modification de variables d'environnement
  • unset : suppression de variables d'environnement
  • env : affichage de l'environnement
  • cd : changement de répertoire
  • pwd : affichage du répertoire courant
  • echo : affichage de texte
  • exit : sortie du shell (ne retourne pas)

Retourne 1 si la commande n'est pas reconnue.

int ft_is_builtin ( char ** cmd,
char *** envp )

Vérifie si une commande est un builtin du shell.

Paramètres
cmdTableau des arguments de la commande
envpPointeur vers le tableau des variables d'environnement (non utilisé)
Renvoie
int 1 si c'est un builtin, 0 sinon

Cette fonction vérifie si le premier argument correspond à l'une des commandes intégrées du shell. Liste des builtins reconnus :

  • echo : affichage de texte
  • cd : changement de répertoire
  • pwd : répertoire courant
  • export : variables d'environnement
  • unset : suppression de variables
  • env : affichage environnement
  • exit : sortie du shell

Retourne 0 si cmd est NULL ou vide.

static int is_invalid_exit_arg ( char ** cmd)
static

Vérifie si les arguments de exit contiennent des caractères invalides.

Paramètres
cmdTableau des arguments de la commande
Renvoie
int 1 si arguments invalides, 0 sinon

Cette fonction vérifie que les arguments de exit sont des nombres valides. Elle ignore les espaces en début et autorise les signes + et - en première position. Retourne 1 si elle trouve des lettres ou caractères non numériques.
