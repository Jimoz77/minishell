int ft_pwd ( void )

Implémente la commande pwd (print working directory)

Renvoie
int 0 en cas de succès, 1 si erreur

Cette fonction affiche le répertoire de travail courant. Utilise getcwd() pour obtenir le chemin absolu et l'affiche sur la sortie standard suivi d'un retour à la ligne.

En cas d'erreur de getcwd(), affiche un message d'erreur et retourne 1.
