Exécute une commande externe avec gestion complète.

Paramètres
cmdTableau des arguments de la commande
envpTableau des variables d'environnement
shellStructure shell
Renvoie
int Code de retour (127 si non trouvé, 126 si erreur)

Processus d'exécution :

  1. Gère les commandes spéciales
  2. Résout le chemin (direct ou via PATH)
  3. Vérifie les permissions et l'existence
  4. Exécute avec execve() Cette fonction ne retourne qu'en cas d'erreur (execve ne retourne jamais si succès).

Exécute uniquement les redirections sans commande.

Paramètres
nodeNœud contenant les redirections
envpPointeur vers l'environnement (non utilisé)
shellStructure shell
Renvoie
int 0 si succès, 1 si erreur

Traite le cas où seules des redirections sont présentes :

  1. Traite tous les heredocs
  2. Applique les redirections
  3. Propage les redirections aux nœuds enfants si présents
  4. Nettoie et restaure l'état

int handle_exec_status ( int status)

Traite le code de retour d'un processus exécuté

Paramètres
statusCode de statut retourné par waitpid()
Renvoie
int Code de retour normalisé

Analyse le statut de sortie :

  • Si terminé par signal : retourne 128 + numéro du signal
  • Si SIGQUIT : affiche "Quit (core dumped)"
  • Si sortie normale : retourne le code de sortie
  • Par défaut : retourne 1
