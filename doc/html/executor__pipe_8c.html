int execute_pipe_node ( t_node * node,
char *** envp,
t_shell * shell )

Exécute un nœud de type pipe.

Paramètres
nodeNœud pipe à exécuter
envpPointeur vers l'environnement
shellStructure shell
Renvoie
int Code de retour du processus droit (dernière commande du pipe)

Orchestration complète d'un pipe :

  1. Vérifie la validité des nœuds gauche et droit
  2. Crée le pipe
  3. Configure les signaux d'exécution
  4. Fork les deux processus enfants
  5. Ferme les descripteurs dans le parent
  6. Attend les deux processus (priorité au droit)
  7. Restaure les signaux et retourne le code du processus droit
