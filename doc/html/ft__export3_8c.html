◆ ft_array_add()

char ** ft_array_add ( char ** array,
const char * str )

Ajoute une chaîne à un tableau de chaînes en réallouant la mémoire.

Paramètres
arrayTableau de chaînes existant (terminé par NULL)
strChaîne à ajouter au tableau
Renvoie
char** Nouveau tableau agrandi, NULL si échec d'allocation

Cette fonction :

  1. Compte les éléments du tableau existant
  2. Alloue un nouveau tableau avec une case supplémentaire
  3. Copie tous les pointeurs existants
  4. Ajoute la nouvelle chaîne (dupliquée)
  5. Termine par NULL et libère l'ancien tableau

Utilisée pour étendre dynamiquement le tableau envp lors d'ajouts de variables.

◆ is_valid_id()

int is_valid_id ( char * str)

Vérifie si une chaîne est un identifiant valide pour une variable d'environnement.

Paramètres
strChaîne à valider
Renvoie
int 1 si valide, 0 sinon

Un identifiant valide doit :

  • Commencer par une lettre (a-z, A-Z) ou un underscore '_'
  • Contenir uniquement des lettres, chiffres et underscores
  • Ne pas être NULL ou vide

Utilisé par export pour valider les noms de variables avant création/modification.
