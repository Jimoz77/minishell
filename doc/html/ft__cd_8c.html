◆ ft_cd()

int ft_cd ( char ** cmd,
char *** envp )

Implémente la commande cd (change directory)

Paramètres
cmdTableau des arguments de la commande (cmd[0] = "cd")
envpPointeur vers le tableau des variables d'environnement
Renvoie
int Code de retour (0 = succès, 1 = erreur)

Cette fonction gère les différents cas de la commande cd :

  • Sans argument ou avec "--" : va vers le répertoire HOME
  • Avec argument : délègue à ft_cd2 pour traitement Met automatiquement à jour PWD et OLDPWD dans l'environnement.

char * old_pwd,

Gère les cas spéciaux de la commande cd (option - et répertoires normaux)

Paramètres
cmdTableau des arguments de la commande
envpPointeur vers le tableau des variables d'environnement
old_pwdChemin du répertoire précédent
resultCode de retour initial
Renvoie
int Code de retour (0 = succès, 1 = erreur)

Cette fonction traite deux cas :

  • Si cmd[1] est "-", elle échange le répertoire courant avec OLDPWD
  • Sinon, elle change vers le répertoire spécifié dans cmd[1] Met à jour les variables PWD et OLDPWD dans l'environnement.

void ft_setenv ( char *** envp,
char * var,
char * new_val )

Met à jour ou crée une variable d'environnement.

Paramètres
envpPointeur vers le tableau des variables d'environnement
varNom de la variable à modifier/créer
new_valNouvelle valeur à assigner

Cette fonction recherche une variable existante dans envp. Si elle la trouve, elle remplace sa valeur. La fonction construit une nouvelle entrée au format "VAR=valeur" et libère l'ancienne. Note: Cette fonction ne gère que la mise à jour, pas l'ajout de nouvelles variables.
