◆ is_redirect_node()

int is_redirect_node ( t_node_type type)

Vérifie si un type de nœud est une redirection.

Paramètres
typeType de nœud à vérifier
Renvoie
int 1 si c'est une redirection, 0 sinon

Identifie les types de nœuds correspondant aux redirections :

  • NODE_REDIRECT_IN : redirection d'entrée (<)
  • NODE_REDIRECT_OUT : redirection de sortie (>)
  • NODE_APPEND : redirection d'ajout (>>)
  • NODE_HEREDOC : here document (<<)

Restaure les descripteurs stdin/stdout originaux :

  1. Utilise dup2() pour restaurer les descripteurs
