static int should_return_current_token ( t_search_context var)
static

Détermine si le token actuel doit être retourné selon les critères.

Paramètres
varStructure contenant les critères de recherche
Renvoie
int 1 si le token doit être retourné, 0 sinon

Critères de sélection :

  • Le token doit être de type TOKEN_WORD
  • Si après un opérateur logique : position doit correspondre au compteur
  • Sinon : la valeur doit correspondre exactement à cmd_value
