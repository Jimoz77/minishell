◆ ft_export()

int ft_export ( char ** cmd,
char *** envp )

Implémente la commande export pour créer/modifier des variables d'environnement.

Paramètres
cmdTableau des arguments de la commande (cmd[0] = "export")
envpPointeur vers le tableau des variables d'environnement
Renvoie
int Code de retour (0 = succès, 1 = erreur d'identifiant, 2 = option invalide)

Cette fonction gère plusieurs cas :

  • Sans argument : affiche l'environnement trié avec declare -x
  • Avec argument : vérifie la validité du nom de variable et met à jour/crée la variable

Règles de validation :

  • Le nom doit commencer par une lettre ou ''
  • Peut contenir des lettres, chiffres et ''
  • Retourne 2 si l'argument commence par '-' (option invalide)
  • Retourne 1 pour les autres identifiants invalides

static char * get_var_name ( const char * arg)
static

Extrait le nom de la variable d'un argument export.

Paramètres
argArgument de la forme "VAR=valeur" ou "VAR"
Renvoie
char* Nom de la variable alloué dynamiquement, NULL si erreur

Cette fonction extrait la partie nom d'une variable d'environnement. Si l'argument contient un '=', elle retourne tout ce qui précède. Sinon, elle retourne l'argument complet. Le résultat doit être libéré.
