Crée un token unique à partir d'un argument de commande.

Paramètres
cmd_argArgument de commande à transformer en token
shellStructure shell pour les informations contextuelles
iPosition du token dans la séquence
Renvoie
t_token* Nouveau token alloué, NULL si erreur

Crée un nouveau token de type TOKEN_WORD avec :

  • Copie de l'argument comme valeur
  • Configuration des parties selon le contexte
  • Initialisation de tous les champs En cas d'échec d'allocation, libère la mémoire et retourne NULL.
