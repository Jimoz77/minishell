t_token * create_tokens_from_cmd ( char ** cmd,
t_shell * shell )

Crée une liste de tokens à partir d'un tableau de commandes.

Paramètres
cmdTableau des arguments de commande
shellStructure shell pour le contexte
Renvoie
t_token* Liste chaînée de tokens, NULL si erreur

Pour chaque argument de commande :

  1. Crée un nouveau token
  2. Trouve et copie les parties du token original correspondant
  3. Lie le token à la liste Libère tout en cas d'erreur d'allocation.

t_shell * shell )

Exécute une commande avec gestion des redirections via fork.

Paramètres
nodeNœud de commande à exécuter
envpTableau des variables d'environnement
shellStructure shell
Renvoie
int Code de retour de l'exécution

Processus sécurisé :

  1. Configure les signaux pour l'exécution
  2. Fork un processus enfant
  3. L'enfant applique les redirections et exécute la commande
  4. Le parent attend et traite le code de retour Gère les signaux et restaure l'état après exécution.
