Étend les tokens d'une commande en utilisant les tokens originaux.

Paramètres
nodeNœud de commande à traiter
shellStructure shell avec les tokens
Renvoie
int 0 si succès, -1 si aucun token trouvé

Processus d'expansion optimisé :

  1. Trouve les tokens correspondants au nœud
  2. Applique l'expansion des variables
  3. Traite les guillemets
  4. Met à jour la commande avec les valeurs finales Préserve les tokens originaux du shell.

Étend les variables d'environnement dans une liste de tokens.

Paramètres
cmd_tokensListe de tokens à traiter
shellStructure shell avec l'environnement

Parcourt les tokens jusqu'à rencontrer un opérateur et applique l'expansion des variables d'environnement sur chaque token de type WORD. S'arrête dès qu'un opérateur est détecté pour éviter de traiter les commandes suivantes.
