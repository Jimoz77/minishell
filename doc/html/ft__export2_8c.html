Copie les pointeurs envp dans la structure t_printsort.

Paramètres
vPointeur vers la structure t_printsort à remplir
envpTableau des variables d'environnement source
Renvoie
int 1 en cas de succès, 0 si échec d'allocation mémoire

Alloue un tableau de pointeurs et copie les références vers les chaînes d'environnement. Ne duplique pas les chaînes elles-mêmes, seulement les pointeurs pour économiser la mémoire. Le tableau résultant est terminé par NULL pour compatibilité avec les fonctions standard.

En cas d'échec d'allocation, retourne 0 et laisse v->envp_cpy non initialisé.

int print_sorted_env ( char ** envp)

Affiche l'environnement trié au format declare -x.

Paramètres
envpTableau des variables d'environnement à afficher
Renvoie
int 1 en cas de succès, 0 si échec d'allocation mémoire

Fonction principale qui orchestre l'affichage trié de l'environnement :

  1. Copie les pointeurs envp dans une structure temporaire
  2. Trie alphabétiquement les pointeurs
  3. Affiche au format bash declare -x
  4. Libère la mémoire temporaire

Utilisée par export sans arguments pour afficher l'état actuel de l'environnement.

static void sort_envp_cpy ( t_printsort * v)
static

Trie le tableau envp_cpy par ordre alphabétique des noms de variables.

Paramètres
vPointeur vers la structure t_printsort contenant le tableau à trier

Utilise l'algorithme de tri par sélection pour ordonner alphabétiquement les pointeurs vers les chaînes d'environnement. Pour chaque position i, trouve l'élément minimal dans la partie non triée [i+1..n] et l'échange avec l'élément en position i. Le tri se base uniquement sur les noms des variables (partie avant '=').

Complexité : O(n²) où n est le nombre de variables d'environnement.

Définition à la ligne 98 du fichier ft_export2.c.

static void swap_if_needed ( t_printsort * v)
static

Échange deux éléments du tableau si nécessaire selon l'ordre alphabétique.

Paramètres
vPointeur vers la structure t_printsort contenant les données de tri

Cette fonction compare deux variables d'environnement pour déterminer leur ordre alphabétique. La comparaison se fait uniquement sur le nom de la variable (partie avant le '='). Algorithme de comparaison :

  1. Extrait les noms des variables (avant '=')
  2. Compare sur la longueur du plus court nom
  3. Si égaux, privilégie le nom le plus court
  4. Échange les pointeurs si l'ordre est incorrect

Utilisée par l'algorithme de tri par sélection dans sort_envp_cpy().
