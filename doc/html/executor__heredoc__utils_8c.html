int apply_heredoc_redir ( t_node * node,
char * delimiter,
t_redirect * red )

Applique une redirection heredoc en trouvant le descripteur correspondant.

Paramètres
nodeNœud contenant la liste des heredocs
delimiterDélimiteur du heredoc à appliquer
redStructure de redirection à configurer
Renvoie
int 1 si succès, 0 si heredoc non trouvé

Recherche dans la liste des heredocs du nœud celui qui correspond au délimiteur spécifié :

  1. Ferme le descripteur d'entrée précédent si ouvert
  2. Parcourt la liste des heredocs du nœud
  3. Compare les délimiteurs de manière exacte
  4. Vérifie que le heredoc a été traité et a un descripteur valide
  5. Configure red->stdin_fd avec le descripteur trouvé

Affiche un avertissement si aucun heredoc correspondant n'est trouvé.

◆ string_to_fd()

int string_to_fd ( const char * content)

Convertit une chaîne en descripteur de fichier via un pipe.

Paramètres
contentChaîne à convertir
Renvoie
int Descripteur de fichier en lecture, -1 si erreur

Crée un pipe et écrit le contenu dans l'extrémité d'écriture, puis ferme cette extrémité et retourne l'extrémité de lecture. Utilisé pour simuler un fichier à partir d'une chaîne en mémoire. En cas d'erreur d'écriture, ferme les deux extrémités et retourne -1.
