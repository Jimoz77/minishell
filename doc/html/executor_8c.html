Exécute un nœud selon son type spécifique.

Paramètres
nodeNœud à exécuter
envpPointeur vers l'environnement
shellStructure shell
Renvoie
int Code de retour de l'exécution

Dispatche l'exécution vers la fonction appropriée :

  • NODE_CMD : execute_cmd_node()
  • NODE_PAREN : execute_paren_node()
  • NODE_PIPE : execute_pipe_node()
  • NODE_AND : execute_and_node()
  • NODE_OR : execute_or_node() Retourne 1 pour les types non reconnus.
