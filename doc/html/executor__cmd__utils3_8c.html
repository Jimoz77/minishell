Exécute une commande externe avec gestion d'erreurs.

Paramètres
cmdTableau des arguments de la commande
envpTableau des variables d'environnement
Renvoie
int Code de retour (127 si non trouvé, 126 si erreur, 2 pour cas spéciaux)

Processus d'exécution :

  1. Gère les commandes spéciales (".", "..")
  2. Recherche le chemin de l'exécutable
  3. Exécute avec execve() Attention : cette fonction appelle execve() et ne retourne qu'en cas d'erreur.

Traite l'expansion des variables dans les tokens de commande.

Paramètres
temp_tokensListe de tokens temporaires à traiter
shellStructure shell avec l'environnement
Renvoie
int 1 en cas de succès

Pour chaque token qui n'est pas entre guillemets simples :

  1. Applique l'expansion des variables d'environnement
  2. Construit la valeur finale sans guillemets si nécessaire Modifie les tokens en place et libère les anciennes valeurs.
