int ft_unset ( char ** cmd,
char *** envp )

Implémente la commande unset pour supprimer des variables d'environnement.

Paramètres
cmdTableau des arguments de la commande (cmd[0] = "unset")
envpPointeur vers le tableau des variables d'environnement
Renvoie
int Toujours 0 (la commande continue même en cas d'erreur sur un argument)

Cette fonction traite tous les arguments fournis à unset :

  • Vérifie la validité de chaque nom de variable
  • Affiche un message d'erreur pour les identifiants invalides
  • Supprime les variables valides trouvées dans l'environnement
  • Continue le traitement même si certains arguments sont invalides

Comportement bash-compatible : n'interrompt pas le traitement sur erreur.

◆ is_valid_idd()

static int is_valid_idd ( char * str)
static

Vérifie si une chaîne est un identifiant valide pour unset.

Paramètres
strChaîne à valider
Renvoie
int 1 si valide, 0 sinon

Un identifiant valide pour unset doit :

  • Commencer par une lettre (a-z, A-Z)
  • Contenir uniquement des lettres, chiffres et underscores
  • Ne pas être NULL

Note: Plus restrictif qu'export (ne permet pas '_' en première position).

static

Supprime une variable d'environnement du tableau envp.

Paramètres
varNom de la variable à supprimer
envpPointeur vers le tableau des variables d'environnement

Cette fonction :

  1. Parcourt envp pour trouver la variable correspondante
  2. Si trouvée, libère la mémoire de l'entrée
  3. Décale tous les éléments suivants d'une position vers la gauche
  4. Met NULL à la dernière position
