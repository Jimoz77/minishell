Paramètres

envpPointeur vers l'environnement
shellStructure shell
Renvoie
int Code de retour de l'exécution

Crée un environnement isolé pour l'exécution :

  1. Duplique l'environnement actuel
  2. Configure shell->envp temporairement
  3. Exécute l'AST du nœud enfant
  4. Restaure l'environnement original
  5. Libère l'environnement temporaire Utilisé quand les parenthèses n'ont pas de redirections.

Fork et exécute un nœud parent avec gestion des signaux.

Paramètres
nodeNœud parent à exécuter
envpPointeur vers l'environnement
shellStructure shell
Renvoie
int Code de retour normalisé

Gère l'exécution dans un processus séparé :

  1. Configure les signaux d'exécution
  2. Fork un processus enfant
  3. L'enfant exécute le contenu des parenthèses
  4. Le parent attend et traite le code de retour
  5. Restaure les signaux originaux Traite les signaux (SIGQUIT affiche "Quit (core dumped)").
